## オブジェクト

### 作成

```ruby
  class Hoge
  end

  # 引数が必要な場合は引数を渡す
  hoge = Hoge.new();
```

### クラスの確認

```ruby
  hoge.class
```

### オブジェクトの id

```ruby
  hoge.object_id
```

## クラス

### 作成

```ruby
  class Hoge
  end
```

### initialize(コンストラクタ)

```ruby
  class Hoge
    def initialize(hoge)
      @hoge = hoge
    end
  end
```
